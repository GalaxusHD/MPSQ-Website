Admin – Inhalte

Bearbeite die Inhalte von „Über uns" und „Event" direkt im Browser.

📋 So funktioniert der Editor:
  1. Wähle oben einen Tab (Über uns oder Event).
  2. Bearbeite die Felder nach Wunsch.
  3. Klicke auf JSON herunterladen – die Datei wird auf deinen Computer gespeichert.
  4. Öffne auf GitHub die Datei data/ueber-uns.json, data/event.json bzw. data/team.json.
  5. Klicke auf den Stift-Button (✏️ „Edit this file"), füge den Inhalt der heruntergeladenen Datei ein und bestätige mit Commit changes.
  6. GitHub Pages aktualisiert sich automatisch innerhalb weniger Minuten.
