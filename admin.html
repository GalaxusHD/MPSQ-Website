Admin – Inhalte

Bearbeite und veröffentliche Inhalte über die Worker API.

📋 So funktioniert der Editor:
  1. Gib oben das Admin-Passwort ein.
  2. Wähle einen Tab (Über uns, Event, Team oder Sozial Media).
  3. Bearbeite die Felder nach Wunsch.
  4. Klicke auf Speichern – Änderungen werden sofort über die Worker API veröffentlicht.
  5. Die Seiten zeigen Änderungen nach einem Reload.

Optional: Mit JSON herunterladen kannst du eine lokale Kopie erstellen.
